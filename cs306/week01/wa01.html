For each of the following functions, indicate how much the function's value will change if its argument is increased fourfold.

a. \(log_2\ n\)

Value change: \(\frac{\log_2 4n}{\log_2 n} = \log_2 4n - \log_2 n = (\log_2 4 + \log_2 n) - \log_2 n = 2\)

b. \(\sqrt{n}\)

Value change: \(\frac{\sqrt{4n}}{\sqrt{n}} = \frac{2\sqrt{n}}{\sqrt{n}} = \frac{2}{1} = 2\)

c. \(n\)

Value change: \(\frac{4n}{n} = \frac{4}{1} = 4\)

d. \(n^2\)

Value change: \(\frac{(4n)^2}{n^2} = \frac{4^2 * n^2}{n^2} = 4^2 = 16\)

e. \(n^3\)

Value change: \(\frac{(4n)^3}{n^3} = \frac{4^3 * n^3}{n^3} = 4^3 = 64\)

f. \(2^n\)

Value change: \(\frac{2^{4n}}{2^n} = 2^{3n}\)
